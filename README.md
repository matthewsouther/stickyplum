# stickyplum

Code repo for stickyplum.com

## Dev requirements

- Recent version of Node.js and NPM

## Setup

Run `npm install` in the terminal to install dependencies.

Run `npm run dev` to run Vite. Vite will display the localhost HTTP address where the website is being served locally. Open the link to view the website in development. Saving key files will trigger a refresh in the browser.
